# Dhairya Portfolio

A modern, responsive portfolio website built with React, TypeScript, and Vite. Features fluid animations, liquid glass effects, and a clean design aesthetic.

## Tech Stack

- **React 18** with TypeScript
- **Vite** for fast development and builds
- **Framer Motion** for smooth animations
- **Tailwind CSS** for styling
- **Lucide React** for icons
- **EmailJS** for contact form

## Features

- Responsive design for all devices
- Animated liquid blob background
- Interactive mouse tracer effect
- Expertise accordion with image gallery
- Project showcase grid
- Contact form with email integration
- Scroll-to-top button

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Deployment

This project is configured for deployment on platforms like Vercel, Netlify, or any static hosting service.

```bash
npm run build
```
